# Epic5 Store

Epic5 Store plugin for Websoft9 Cockpit

## Version History

### [0.1.0] - 2026-02-08

#### Added
- Initial release of Epic5 Store plugin
- Media data display with categorization
- Responsive grid layout with Material-UI
- Category-based filtering
- Decoupled from apphub for independent operation
- Support for logos, screenshots, and media metadata
- Card-based UI for media items

#### Features
- Browse media resources by category
- Search and filter functionality
- Responsive design for all screen sizes
- Standalone Cockpit plugin architecture
