# Websoft9 手动测试指南

> 📖 **面向人群：** 非专业测试人员、社区贡献者  
> 🎯 **目标：** 在发布新版本前，确保所有核心功能正常工作  
> ⏱️ **预计耗时：** 30-45 分钟（全新安装）/ 15-20 分钟（升级测试）

---

## 🚀 快速开始

### 准备工作

1. **准备一台测试服务器**（建议 Ubuntu 24.04 或 CentOS Stream 9）
2. **安装 Websoft9**（全新安装或升级现有安装）
3. **打开浏览器**访问 `http://服务器IP:9000`
4. **打开测试清单 Issue**，对照逐项测试

### 安装命令

```bash
# 全新安装
wget -O install.sh https://artifact.websoft9.com/websoft9/release/platform/install.sh && sudo bash install.sh

# 升级已有安装
wget -O install.sh https://artifact.websoft9.com/websoft9/release/platform/install.sh && sudo bash install.sh --version "latest"
```

## 📋 各模块测试详解

### 1. 系统概览

| 测试项 | 操作步骤 | 预期结果 |
|--------|----------|----------|
| 系统信息展示 | 登录后查看首页 | 看到 CPU 使用率、内存使用率、磁盘空间、网络流量等图表 |
| 数据刷新 | 等待 10-20 秒 | 图表数据自动更新，无卡死 |
| 容器状态 | 查看概览中的容器列表 | 四个容器显示绿色运行状态 |

### 2. 应用商店

| 测试项 | 操作步骤 | 预期结果 |
|--------|----------|----------|
| 应用列表加载 | 点击左侧菜单「应用商店」 | 页面显示应用卡片列表 |
| 分类筛选 | 点击顶部分类标签（如 CMS） | 只显示该分类下的应用 |
| 应用搜索 | 在搜索框输入「WordPress」 | 搜到 WordPress 相关应用 |
| 应用详情 | 点击某个应用卡片 | 弹出详情窗口，有描述和截图 |
| 一键安装 | 在详情中点击「安装」按钮 | 开始安装，有进度提示 |
| 安装完成 | 等待安装结束 | 提示安装成功，应用出现在「我的应用」 |

### 3. 我的应用

| 测试项 | 操作步骤 | 预期结果 |
|--------|----------|----------|
| 应用列表 | 点击左侧菜单「我的应用」 | 显示所有已安装应用 |
| 启动应用 | 对已停止的应用点「启动」 | 状态变为运行中 |
| 停止应用 | 对运行中的应用点「停止」 | 状态变为已停止 |
| 重启应用 | 点「重启」 | 应用重新启动成功 |
| 查看日志 | 点「日志」按钮 | 显示实时日志，错误红色高亮 |
| 环境变量 | 点「环境变量」标签 | 能查看和编辑变量 |
| 重新部署 | 点「重新部署」 | 应用重新部署成功 |
| 卸载应用 | 点「卸载」，确认 | 应用从列表消失 |

### 4. 反向代理 & SSL

| 测试项 | 操作步骤 | 预期结果 |
|--------|----------|----------|
| 查看代理列表 | 进入 Nginx/代理管理页面 | 显示已有代理列表 |
| 创建代理 | 点「添加」，输入域名和端口 | 代理创建成功 |
| 申请 SSL | 在代理设置中开启 SSL，选 Let's Encrypt | 证书申请成功 |
| HTTPS 访问 | 用 https://域名 访问 | 显示安全锁图标 |
| 强制 HTTPS | 开启强制 HTTPS 开关 | HTTP 自动跳转到 HTTPS |

> ⚠️ Let's Encrypt 证书申请需要域名正确解析到服务器 IP

### 5. 文件管理

| 测试项 | 操作步骤 | 预期结果 |
|--------|----------|----------|
| 目录浏览 | 点击「文件管理」 | 显示文件目录树 |
| 上传文件 | 点「上传」，选择文件 | 文件上传成功 |
| 下载文件 | 选中文件，点「下载」 | 文件下载到本地 |
| 新建文件夹 | 点「新建文件夹」 | 文件夹创建成功 |
| 重命名 | 右键文件，选「重命名」 | 名称修改成功 |
| 删除 | 右键文件，选「删除」 | 文件被删除 |
| 编辑文件 | 双击文本文件 | 在线编辑器打开，能编辑保存 |
| 修改权限 | 右键文件，修改权限 | 权限修改成功 |

## 🖼️ 截图指南

### 如何截图

- **Windows：** `Win + Shift + S`（截图工具）或 `PrtSc` 键
- **Mac：** `Cmd + Shift + 4`（选区截图）
- **浏览器截图：** Chrome/Edge 按 `F12` → `Ctrl+Shift+P` → 输入 "screenshot"

### 什么情况需要截图

- ✅ 功能正常时的界面（作为通过证据）
- ❌ 出现错误提示或异常界面
- 🐛 操作结果与预期不符
- 📊 数据展示异常

## 🔧 常见问题

### Q: 容器显示异常状态怎么办？
```bash
# 在服务器终端检查容器状态
docker ps -a
# 查看容器日志
docker logs websoft9-apphub
```

### Q: 应用安装失败？
检查：
1. 服务器磁盘空间是否充足（`df -h`）
2. Docker 服务是否正常（`systemctl status docker`）
3. 网络连接是否正常

### Q: SSL 证书申请失败？
1. 确认域名已正确解析到服务器 IP
2. 确认 80 端口可以从外网访问
3. 检查防火墙设置
