# Epics

Epics 设计文档，每个 Epic 对应一个功能模块

## 目录结构

- **app-management-epic.md** - 应用管理模块
- **proxy-management-epic.md** - 反向代理管理模块  
- **system-settings-epic.md** - 系统设置模块
- **backup-restore-epic.md** - 备份恢复模块

## 说明

每个 Epic 文档包含：
1. Epic 概述（业务目标、核心价值、验收标准）
2. 技术规范（架构设计、API 端点、数据模型）
3. Stories 拆分（可执行的开发任务）
4. 依赖关系
5. 风险与挑战
6. 测试策略
7. 监控指标
